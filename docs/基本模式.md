## 插入模式 - 命令模式

| 按键操作 | 用途                  |
| -------- | -------------------- |
| Esc      | 返回普通模式          |
| Ctrl + [ | 返回普通模式          |
| Ctrl + o | 切换到“插入-普通模式”  |
